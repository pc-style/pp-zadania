## Zadanie 1.
Celem rozwiązania tego zadania jest zdobycie umiejętności przekazywania argumentów funkcji poprzez wartość, referencję i wskaźnik.

*   Zaprojektuj i zaimplementuj trzy funkcje `suma` zwracające sumę liczb całkowitych od 1 do liczby `n` podanej przez użytkownika i przekazanej do tych funkcji na trzy sposoby: przez wartość, referencję i wskaźnik. Liczbę `n` należy pobierać od użytkownika dopóki nie poda wartości większej od 2.
*   Zaprojektuj i zaimplementuj funkcję `czy_parzysta` sprawdzającą, czy liczba przekazana do niej poprzez wskaźnik jest parzysta. Funkcja zwraca wartość logiczną.
*   Zaprojektuj i zaimplementuj funkcję `int potega(const int a, const int n)` zwracającą n-tą potęgę liczby `a` (bez korzystania z funkcji bibliotecznej `pow()`).
*   Zaprojektuj i zaimplementuj funkcję `suma_cyfr` zwracającą sumę cyfr w liczbie `n` przekazanej jako argument tej funkcji poprzez referencję.
*   Zaprojektuj i zaimplementuj funkcję `int suma_parzystych(const int a, const int b)` zwracającą sumę parzystych liczb z przedziału `⟨a, b⟩`.
*   Zaprojektuj i zaimplementuj funkcję `kod_ascii` zwracającą kod ASCII znaku podanego przez użytkownika i przekazanego do funkcji jako argument.
*   Zaprojektuj i zaimplementuj funkcję `main` zapewniającą prawidłowe działanie programu, w którym wywołane zostaną w/w funkcje. Wartości zwracane przez funkcje należy wypisać na standardowym wyjściu (monitorze).

## Zadanie 3.
Celem rozwiązania tego zestawu zadań jest utworzenie symulatora kalkulatora. Do rozwiązania nie wolno używać zmiennych globalnych oraz funkcji bibliotecznych (np. do potęgowania). Każda funkcja stanowi jedną opcję do wyboru podczas obsługi kalkulatora. Argumenty formalne funkcji należy przekazywać w sposób odpowiedni do potrzeb danej funkcji. Należy pamiętać o używaniu modyfikatora `const` tam, gdzie jest to potrzebne. Symulator ma działać analogicznie do rzeczywistego kalkulatora tzn. po uruchomieniu działa dotąd dopóki użytkownik nie przerwie jego działania.

*   Zaprojektuj i zaimplementuj funkcję `suma` obliczającą sumę dwóch liczb przekazanych, jako parametry oraz zwracającą wynik tej operacji.
*   Zaprojektuj i zaimplementuj funkcję `roznica` obliczającą różnicę dwóch liczb przekazanych, jako parametry oraz zwracającą wynik tej operacji.
*   Zaprojektuj i zaimplementuj funkcję `iloczyn` obliczającą iloczyn dwóch liczb przekazanych, jako parametry oraz zwracającą wynik tej operacji.
*   Zaprojektuj i zaimplementuj funkcję `iloraz` obliczającą iloraz dwóch liczb przekazanych, jako parametry oraz zwracającą wynik tej operacji. Należy pamiętać o sprawdzeniu poprawności matematycznej takiego dzielenia.
*   Zaprojektuj i zaimplementuj funkcję `kwadrat` obliczającą kwadrat liczby przekazanej, jako parametr oraz zwracającą wynik tej operacji. Nie wolno używać funkcji `pow()`.
*   Zaprojektuj i zaimplementuj funkcję `szescian` obliczającą sześcian liczby przekazanej, jako parametr oraz zwracającą wynik tej operacji. Nie wolno używać funkcji `pow()`.
*   Zaprojektuj i zaimplementuj funkcję `wartosc_wyrazenia` obliczającą wartość poniższego wyrażenia dla liczby przekazanej, jako parametr oraz zwracającą wynik tej operacji.
    `f(x) = (10 · x^3 + 3.1415 · x^2) · (x^3 - 1 / x^2)`
*   Zaprojektuj i zaimplementuj funkcję `obwod` obliczającą obwód trójkąta dla długości boków przekazanych, jako parametry oraz zwracającą wynik tej operacji. W funkcji należy sprawdzić, czy z podanych odcinków można zbudować trójkąt.
*   Zaprojektuj i zaimplementuj funkcję `pole` obliczającą pole trójkąta metodą Herona dla długości boków przekazanych, jako parametry oraz zwracającą wynik tej operacji. W zadaniu należy wykorzystać funkcję `obwod` zwracającą obwód trójkąta.
*   Zaprojektuj i zaimplementuj funkcję `menu`, z wykorzystaniem instrukcji wyboru `switch`, symulującą działanie kalkulatora oferującego działania określone wszystkimi wcześniej zaimplementowanymi funkcjami. Program powinien posiadać menu umożliwiające wybór operacji do wykonania (wszystkie wcześniej napisane funkcje + opcja zakończenia działania programu). Funkcja ma umożliwić działanie kalkulatora tak długo, aż użytkownik nie wybierze opcji kończącej jego działanie.

Można tworzyć dodatkowe funkcje oprócz wymienionych powyżej, ale nie można nic dodać w funkcji `main()`.

Zadanie przetestować poniższym fragmentem kodu źródłowego:

```c
int main()
{
    while(kalkulator());
    return 0;
}
```

## Zadanie 4.
Celem zadania jest nabycie umiejętności pracy z automatyczną tablicą jednowymiarową oraz przekazywaniem i używaniem argumentów funkcji przekazanych poprzez wartość, referencję i wskaźnik.

*   Zaprojektuj i zaimplementuj funkcję `void wypisz(const int* tab, const int n);` wypisującą na standardowym wyjściu elementy tablicy `tab` przekazanej do tej funkcji poprzez wskaźnik.
*   Zaprojektuj i zaimplementuj funkcję `void wypelnij(int* tab, const int n);` wypełniającą tablicę `tab` w następujący sposób:
    *   dla każdego parzystego indeksu `i` wstawia wartość silni z `i`,
    *   w przeciwnym wypadku wstawia wartość i-tego elementu ciągu Fibonacciego ze znakiem minus.
*   Zaprojektuj i zaimplementuj funkcję `void wypelnij(int* tab, const int n, const int a, const int b);` wypełniającą tablicę `tab` całkowitymi liczbami pseudolosowymi z zakresu `⟨a, b⟩` podanego przez użytkownika.
*   Zaprojektuj i zaimplementuj funkcję `suma` zwracającą sumę dwóch tablic jednowymiarowych wypełnionych za pomocą w/w funkcji:
    *   `int* suma1(const int* tab1, const int* tab2, const unsigned int n);`
    *   `void suma2(const int* tab1, const int* tab2, int*& tab3, const unsigned int n);`
    *   `void suma3(const int* tab1, const int* tab2, int* tab3, const unsigned int n);`
*   Zaprojektuj i zaimplementuj funkcję `tabliczka_mnozenia` wyświetlającą na standardowym wyjściu (monitorze) tabliczkę mnożenia od 1 do 100 oraz zwracający iloczyn dwóch wybranych przez użytkownika liczb całkowitych z zakresu `⟨1, 10⟩`.
*   Zaprojektuj i zaimplementuj funkcję `main` zapewniającą prawidłowe działanie programu, w którym wywołane zostaną w/w funkcje.
